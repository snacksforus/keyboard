# Keyboard Configuration

Kinesis Advantage 360 (SmartSet) configuration with Leader Key and Magnet.

## Layout

Hot Keys:

hk3 - Leader Key Shortcut
hk4 - JetBrains Action Menu

Thumb Clusters:

  ┌─┬─┐          ┌─┬─┐
  │⌃│⌘│          │⌘│⌃│
┌─┼─┼─┤          ├─┼─┼─┐
│ │ │⌥│          │⌥│ │ │
│⌫│⌦├─┤          ├─┤↩│␣│
│ │ │↘│          │⇟│ │ │
└─┴─┴─┘          └─┴─┴─┘


Left keywell (application switching using Leader Key):

b - BBEdit
c - Chrome
d - Slack/Signal
e - WebStorm
f - Finder
g - GoLand
s - Safari
t - Terminal

Right keywell (window management and special characters using left function key):

u - open/close parenthesis
i - open/close curly brace
o - open/close square brace

j - tile left half (ctrl func left-arrow)
k - tile right half (ctrl func right-arrow)
; - center (ctrl func C)
l - fill (ctrl func F)

## Configuration

### Magnet

Tile Left Half:    ⌃⌥⇧⌘←
Tile Left Half:    ⌃⌥⇧⌘→
Center Two Thirds: ⌃⌥⇧⌘C
Maximize:          ⌃⌥⇧⌘F

### Leader Key

Shortcut: ⌃⌥⇧⌘L

### System Settings

Use F1, F2, etc. keys as standard function keys: enabled
Sticky Keys:                                     enabled
